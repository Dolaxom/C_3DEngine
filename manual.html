C_3DEngine

This is a School 21 educational project - an implementation of a program to view 3D wireframe models (3D Viewer) in the C programming language with the additional use of OpenGL, Qt and QtGifImage libraries. All the values inserted in C_3DEngine’s input fields while running the application don’t affect the actual model file in an way as C_3DEngine is a program made for the model viewing purposes only.

C_3DEngine provides the ability to:

  1. - view any applicable .obj file in the filesystem that contains at least up to a 1,000,000 polygons;
  2. - parse .obj files with unconvential number of polygons;
  3. - acquire information on the amount of edges and vertices that the model contains;
  4. - change position, rotation and scale of the model;
  5. - switch the view between perspective and orthogonal projections;
  6. - apply several color and style settings to the display field;
  7. - take a screenshot of the display field and save it in .jpg or .bmp image formats;
  8. - record the display field as a .gif (10fps, any length).

Controls:

  1. [left mouse button] - change position of the model on the x, y axis
  2. [right mouse button] - change rotation of the model on the x, y axis
  3. [mouse scroll wheel] - change position of the model on the z axis
  4. [enter] - apply settings and/or redraw
  5. [tab] - switch focus between input fields
  6. [F11] - window fullscreen mode
  7. [escape] - exit
